# SwiftCart

SwiftCart is a frontend e-commerce project built using HTML, CSS, and JavaScript.  
The project demonstrates core frontend concepts such as dynamic rendering, user interaction, and basic state management using browser storage.

The main goal of this project is to practice real-world JavaScript logic and build a functional shopping interface without using any frontend framework.

## Features
- Product listing
- Add to cart functionality
- Remove items from cart
- Cart data persistence using localStorage
- Responsive layout for different screen sizes

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Project Purpose
This project was built for learning and practice purposes to strengthen frontend fundamentals such as:
- DOM manipulation
- Event handling
- Logical thinking in JavaScript
- Structuring frontend projects
- Using Git and GitHub for version control

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a browser  
   or use Live Server in VS Code

No additional setup or installation is required.
